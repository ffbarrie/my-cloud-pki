# OCSP

OCSP responder configuration and service assets.

Per [ADR-0004](https://github.com/ffbarrie/my-cloud/blob/main/docs/adr/0004-ejbca-online-issuing-ca.md),
EJBCA is the system of record for OCSP initially. Use this directory for
publication URLs, reverse-proxy snippets, and any later split-out OCSP service.

See [../issuing-ca/](../issuing-ca/) for the online issuing CA.

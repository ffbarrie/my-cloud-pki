# EST

Enrollment over Secure Transport (EST) service configuration and assets.

Per [ADR-0004](https://github.com/ffbarrie/my-cloud/blob/main/docs/adr/0004-ejbca-online-issuing-ca.md),
EST is configured primarily in EJBCA. Use this directory for endpoint notes,
client examples, and proxy configuration.

See [../issuing-ca/](../issuing-ca/) for the online issuing CA.

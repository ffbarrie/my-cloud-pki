# CRL

CRL publication and distribution configuration and assets.

Per [ADR-0004](https://github.com/ffbarrie/my-cloud/blob/main/docs/adr/0004-ejbca-online-issuing-ca.md),
EJBCA generates CRLs for the issuing CA. Use this directory for distribution
endpoints, static publish targets, and CDN / web-server snippets if CRLs are
mirrored outside EJBCA.

See [../issuing-ca/](../issuing-ca/) for the online issuing CA.
